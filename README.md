# FileUploadJS

file upload web base on nodejs

## Requirement

- Nodejs v16.19.0
- npm 8.19.3

## Fast Start

```shell
git clone https://github.com/FaterYU/FileUploadJS.git
cd FileUploadJS
npm i -s
node server.js
```

## Back Runing

```shell
npm install pm2 -g
cd FileUploadJS
pm2 start server.js --watch
```

## Check status

```shell
pm2 status server
```
